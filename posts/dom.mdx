## **✅ DOM이란**

DOM(Document Object Model)이란 문서 객체 모델로 HTML 태그를 트리 형태로 구조화 시킨 것을 DOM이라고 한다.
DOM의 모든 HTML 태그는 객체이며 태그 내의 문자까지 객체이다.

이런 객체는 자바스크립트를 통해 접근할 수 있으며 페이지를 조작할 때 사용된다.

즉, DOM은 HTML 로 만들어진 웹페이지와 스크립트 언어 또는 프로그래밍 언어를 이어주는 역할을 한다.

자바스크립트에서 DOM에 접근을 할 때는 `document.body` 형태로 접근하며 이는 `<body>` 태그를 의미한다
